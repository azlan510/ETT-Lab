# React Portfolio Integration

## Purpose
This phase integrates an existing React portfolio website
into the Docker + Virtual Machine deployment project.

## Setup Steps
1. Copied React portfolio source code into the `website/` directory.
2. Installed dependencies using `npm install`.
3. Verified development server using `npm start`.
4. Generated production build using `npm run build`.

## Production Build
The production-ready static files are generated inside:
website/build/

These files will be used in the next phase for Docker containerization.

## Benefits
- Real-world project deployment
- Practical React production workflow
- Ready for Nginx-based hosting
