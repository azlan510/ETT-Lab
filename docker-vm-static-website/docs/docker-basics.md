# Docker Basics and Testing

## Basic Docker Commands

- Check Docker version:
  docker --version

- List running containers:
  docker ps

- List all containers:
  docker ps -a

- List Docker images:
  docker images

## Test Docker Installation

To verify Docker installation, the official hello-world image was used.

Command:
docker run hello-world

## Expected Result
Docker downloads the hello-world image and runs a test container,
displaying a confirmation message.

This confirms that Docker is correctly installed and functional.
